Nitron's dotfiles
=================

Installation:

```
cd
git clone git@github.com:Nitron/dotfiles.git .dotfiles
cd .dotfiles
git submodule init
git submodule update
cd
ln -s .dotfiles/.gitconfig .gitconfig
ln -s .dotfiles/.zshrc .zshrc
ln -s .dotfiles/.vimrc .vimrc
ln -s .dotfiles/.inputrc .inputrc
```
